Intro


Jado is a javascript library which makes all objects iterables. In this way one can generically traverse (even recusrively) complex objects or arrays even if specific elements are singletons such as Numbers, Dates, or other objects.

Features

Jado functions:

  • map / filter / reduce applied to any object
  • a better typeof operator
  • pretty printing json
  • counting data set (CSET) - sub object which provides descriptive stats on collections
    • Mean
    • Variance
    • Most Frequent Item
    • Most Frequent Item stats / histogram
    • ...


    Jado has no dependancies can be used along side any other framework.
