Jado Auto Docs Page


Jado is a library for treating any object as an interable list and applying methods to that objects members. If the object is a singleton (such as a number or Date()) then JADO acts like that object is an array of one of type of that object. This allows map, filter, reduce, each to be applied to any oject.




JADO has no dependancies and can be used along side any other framework.
However for most cases more modern treatment of iterables using ES6 Iterable should be used.
